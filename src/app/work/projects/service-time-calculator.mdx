## Visão Geral

Este projeto é uma calculadora de períodos trabalhados por uma pessoa que soma os intervalos entre os períodos para chegar ao tempo de serviço.

## Funcionalidades

- Adicionar períodos
- Remover períodos
- Calcular o tempo trabalhado em anos e meses
- Baixar o resultado do cálculo em PDF ou PNG
- Algumas das funcionalidades podem ser acessadas por atalho:
  - Digite A para Adicionar um período
  - Digite C para Calcular

## Tecnologias Utilizadas

- HTML, CSS e JS
- Moment.js
- FileSaver.js
- jsPDF
- HTML2Canvas
- Sweet Alert
- jQuery Mask
- jQuery
- Bootstrap
