## Visão Geral

Informações sobre transporte público - Reúne informações úteis e importantes para os usuários do transporte público da cidade de Sabará-MG.

## Funcionalidades

- Buscar por uma linha
- Visualizar os horários de partida da linha
- Os quadros de horários podem ter itinerários diferentes, sendo necessário visualizar as observações do horário de partida.
- Visualizar os dados da linha:
- Nome
- Local de partida e destino
- Tarifa
- Tipo (ônibus, executivo, etc.)
- Abrangência (municipal, metropolitano, rodoviário, etc.)
- Se possui integração ou não
- Concessionária
- Visualizar os pontos de recarga da linha
- Visualizar os pontos de parada da linha

## Tecnologias Utilizadas

- **React**: Para aplicações React rápidas e renderizadas no servidor.
- **React-bootstrap**: Para estilização e como padrão de design.
- **Axios, Express, Node.js**: Para conexão, back-end e requisições no back-end.
- **MySQL, MySQL Workbench, PHPMyAdmin**: Banco de dados e SGBD.
- **Figma**: Para criar e gerenciar assets de design e protótipos.
