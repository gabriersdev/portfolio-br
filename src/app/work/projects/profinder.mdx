## Visão Geral

Um portal para consulta de profissionais aliados dentro da comunidade, desenvolvido em PHP com Bootstrap e TailwindCSS.

## Tecnologias Utilizadas

-   **Código:** PHP e JS
-   **Banco de Dados:** MySQL
-   **Frameworks:** JQuery, JQuery Mask, JQuery UI, JQuery Autocomplete e Sweet Alert
-   **Estilização:** Tailwind CSS e Bootstrap
