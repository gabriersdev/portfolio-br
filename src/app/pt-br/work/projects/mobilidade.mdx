## Overview

Information about public transport - Brings together useful and important information for public transport users in the city of Sabará-MG.

## Features

- Search for a line
- View the line's departure times
- The timetables may have different itineraries, so it is necessary to view the departure time notes.
- View the line's data:
- Name
- Departure and destination location
- Fare
- Type (bus, executive, etc.)
- Scope (municipal, metropolitan, highway, etc.)
- Whether it has integration or not
- Concessionaire
- View the line's charging points
- View the line's stopping points

## Technologies Used

- **React**: For fast, server-rendered React applications.
- **React-bootstrap**: For styling and as a design pattern.
- **Axios, Express, Node.js**: For connection, back-end and back-end requests.
- **MySQL, MySQL Workbench, PHPMyAdmin**: Database and DMS.
- **Figma**: For creating and managing design assets and prototypes.

## Next steps

- Develop a dashboard to monitor departures and predict when buses will approach.
- Automate updating of fare values and stops for transport lines.
- Include subway lines, station mapping and integration with system lines.
