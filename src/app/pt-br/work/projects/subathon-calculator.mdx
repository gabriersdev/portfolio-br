## Overview

The project is a calculator where the different forms of contributions to the stream are converted into time that will be added to the subathon timer. Developed with React and based on Bootstrap, the project is originally a fork of the NLW E-sports repository.

## Libraries and Frameworks

- Bootstrap
- Bootstrap Icons
- Moment
- Frame Motion
- React Number Format
