## Visão Geral

O projeto é uma calculadora onde as diferentes formas de contribuição para a stream são convertidas em tempo que será adicionado ao cronômetro da subathon. Desenvolvido com React e baseado em Bootstrap, o projeto é originalmente um fork do repositório NLW E-sports.

## Bibliotecas e Frameworks

- Bootstrap
- Bootstrap Icons
- Moment
- Framer Motion
- React Number Format
