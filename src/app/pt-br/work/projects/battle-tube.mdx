## Visão Geral

Este é um projeto para permitir que os usuários escolham o melhor clipe de um canal da Twitch. Usando dados obtidos através da API da plataforma, os clipes são exibidos em uma lista de pares para que os usuários possam votar no seu favorito.

São seis rodadas no total. Para que funcione corretamente, foi necessário obter 64 clipes. O critério de seleção nesta fase foi a popularidade dos clipes, ou seja, os clipes mais vistos do canal. Após a seleção, todos os clipes foram analisados e aqueles que não eram adequados para o projeto foram removidos e substituídos pelos subsequentes, totalizando 64 clipes.

Posteriormente, será possível adicionar novos clipes e rodadas e o código adaptado para uso por outras comunidades será disponibilizado no repositório original do projeto.
