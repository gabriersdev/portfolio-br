## Overview

This project is a calculator of periods worked by a person that adds the intervals between periods to arrive at the time worked.

## Functionalities

- Add periods
- Remove periods
- Calculate the time worked in years and months
- Download the result of the calculation in PDF or PNG
- Some of the functionalities can be accessed by shortcut:
- Type A to Add a period
- Type C to Calculate

## Technologies Used

- HTML, CSS and JS
- Moment.js
- FileSaver.js
- jsPDF
- HTML2Canvas
- Sweet Alert
- jQuery Mask
- jQuery
- Bootstrap
